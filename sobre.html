Sobre o Projeto

Tecnologias Utilizadas

Este projeto tem como objetivo criar um gerador de imagens a partir de textos usando inteligência artificial. Ele foi idealizado como uma atividade prática para aplicar os conhecimentos adquiridos em sala de aula de forma criativa e funcional.

Finalidade

Incentivar a aprendizagem ativa por meio do desenvolvimento de uma aplicação real, que envolve a construção de uma interface web, comunicação com uma API externa e exibição dinâmica de conteúdos visuais.

O que cada tecnologia faz

HyperText Markup Language

Responsável por estruturar todo o conteúdo da página, incluindo formulários, títulos, parágrafos, links e imagens

Cascading Style Sheets

Responsável por aplicar estilos à interface — cores, tamanhos, espaçamentos, layout responsivo e efeitos visuais

JavaScript

Responsável por adicionar interatividade, captar dados do usuário, enviar requisições para a IA e exibir as imagens geradas dinamicamente na página
